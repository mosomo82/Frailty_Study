# METADATA FOR Frailty Study Dataset

## 1. DATASET DESCRIPTION

This dataset contains information from a study on physical frailty from 10 female participants. It includes various physical measurements data for each participant. Frailty is physical weakness; lack of health or strength. Reduced grip strength in females correlated with higher frailty scores and vice versa. Hand grip strength can be quantified by measuring the amount of static force that the hand can squeeze around a dynamometer

## 3. COLUMN DESCRIPTIONS

This dataset contains 5 columns:

1.  **Height**
    * **Description:** The participant's height.
    * **Data Type:** Integer (Numerical)
    * **Units:** Inches

2.  **Weight**
    * **Description:** The participant's body weight.
    * **Data Type:** Integer (Numerical)
    * **Units:** Pounds

3.  **Age**
    * **Description:** The age of the participant at the time of the study.
    * **Data Type:** Integer (Numerical)
    * **Units:** Years

4.  **Grip_Strength**
    * **Description:** A measure of the participant's hand and forearm strength. This is a common indicator used in frailty assessments.
    * **Data Type:** Integer (Numerical)
    * **Units:** Kilograms (kg)

5.  **Frailty**
    * **Description:** A binary indicator of the participant's frailty status, likely determined by a set of clinical criteria.
    * **Data Type:** Text (Categorical)
    * **Values:** "Y" (Frail), "N" (Not Frail)
